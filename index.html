Kadertjesmaker
Op zoek naar de vaste, niet-aanpasbare sjablonen? Klik dan hier.

1. Klik in het kadertje hieronder, vul tekst in en maak op met de tekstverwerker
• Via de knop met de stempel uiterst links kan je een sjabloon kiezen (bv. kadertje voor assisenstuk)
• Voor een afbeelding moet je bij Source de url invullen, foto's uploaden kan hier in MNT





2. Kies een achtergrondkleur
• Via de testknop hierboven kan je de kleur aanpassen in het voorbeeld
• Kies de definitieve kleur door het juiste bolletje hieronder aan te duiden











3. Kopieer deze code en plak ze als embed in het artikel
• Klik in het vak om alle tekst te selecteren
• Gebruik daarna CTRL + C of rechtermuisknop om de code te kopiëren

Druk op de knop BEVESTIG om de code aan te maken.
